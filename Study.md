#### -자료형

숫자, 문자 등의 자료형태

| 자료형             | Ex            |
| ------------------ | ------------- |
| 정수 ( int )       | 1, 100, 12    |
| 실수( float )      | 3.14 , 314e-2 |
| 복소수 ( complex ) | 3+4j          |
| 불 (bool)          | True, False   |
| 문자 ( str )       | 'Hi'          |
